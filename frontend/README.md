# Dalal Street Edge Frontend

Minimal, clean React + TypeScript + Tailwind CSS frontend for multi-agent Indian stock analyzer.

## Setup

```bash
npm install
npm run dev
```

Opens at `http://localhost:5173`

## Features

- Home/Analyze page with real-time step-by-step loading simulation
- History page with previous analyses
- Mock data with hardcoded results
- Fully responsive design
- All API calls stubbed for backend integration
